Houdini Foundation

기간: 6주

Houdini 입문자를 위한 6주 기초 프로그램입니다. 인터페이스 이해부터 절차적(Procedural) 사고, 기초 시뮬레이션, 간단한 결과물 제작까지 밀도 있게 학습합니다.

학습 목표

  • 노드 기반 작업 방식과 데이터 흐름을 이해한다.
  • 절차적 모델링과 Attribute 제어를 직접 구성한다.
  • 재사용 가능한 씬 구조와 정리된 작업 습관을 만든다.

6주 커리큘럼

  • 1주차: Houdini UI, 노드 그래프 기초, SOP 컨텍스트, 파일 관리
  • 2주차: Procedural Modeling 기본, Group/Transform/Copy 응용
  • 3주차: Attribute 이해, 랜덤/분포 제어, 기본 표현식
  • 4주차: VOP/VEX 입문, 파라미터 기반 셋업 구성
  • 5주차: 기본 시뮬레이션(파티클/RBD 입문), 캐시 개념, 씬 정리
  • 6주차: 파이널 미니 프로젝트 제작, 리뷰, 다음 단계 가이드

최종 산출물

  • 절차적 모델링 결과물 1종
  • 기초 시뮬레이션 샷 1종
  • 노드 구조 캡처 + 간단한 작업 설명 문서

권장 대상

  • Houdini를 처음 시작하는 아티스트/학생
  • Maya/Blender/C4D 경험은 있지만 절차적 워크플로우가 처음인 분
  • Master/Procedural Generation 과정 진입을 준비하는 분
